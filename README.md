# Product Description Matching System

A Streamlit-based application that compares new product descriptions with existing product descriptions and identifies similar records using RapidFuzz.

## Technologies
- Python
- Pandas
- OpenPyXL
- RapidFuzz
- Streamlit

## Features
- Excel upload
- Similarity matching
- Threshold filtering
- Cloud deployment

## Run Locally

pip install -r requirements.txt

python -m streamlit run ui_app.py
